# 🚀 mic-notebook

An open-core, AI-powered interactive notebook extension for VS Code.

---

## ⚡ Overview
**mic-notebook** reimagines the developer workflow by bringing the power of interactive, cell-based coding (like Jupyter Notebooks) directly into VS Code, supercharged with a native, context-aware **AI Agent**. 

Built with an **Open-Core model**, our mission is to provide an elite, free tool for individual developers worldwide, while scaling into an enterprise SaaS that offers secure real-time collaboration for engineering teams.

## ✨ Key Features
*   **Interactive Code Cells:** Run, isolate, and test code blocks seamlessly inside your favorite editor.
*   **Context-Aware AI Agent:** A dedicated AI sidekick that understands your workspace, autofixes runtime errors, and generates clean code.
*   **Lightning Fast:** Built on top of the native VS Code/Code-OSS notebook architecture.
*   **Enterprise Ready (Upcoming):** Real-time collaboration, secure data layers (anti-leakage), and comprehensive audit logs.

## 🛠️ Tech Stack
*   **Extension Layer:** TypeScript, VS Code Extension API, Webview View Providers.
*   **AI Intelligence:** Integrated LLM contexts (supporting Gemini, OpenAI, Anthropic, and local Ollama models).
